## Usage Notes

### Getting Started
Each skill directory contains:
- **SKILL.md** - API documentation with endpoints and examples
- **api.ts** - TypeScript client with typed methods

### Authentication
For APIs requiring keys:
- **OpenWeatherMap** - Sign up at openweathermap.org
- **NASA** - Use `DEMO_KEY` for testing or get key at api.nasa.gov
- **News API** - Free tier available at newsapi.org
- **NYTimes** - Get key at developer.nytimes.com
- **Helius** - Get key at helius.xyz
- **SerpAPI** - Get key at serpapi.com

### Rate Limits
Most public APIs have rate limits:
- JSONPlaceholder: No limits
- GitHub: 60/hour (unauthenticated), 5000/hour (authenticated)
- CoinGecko: 50 calls/minute (free tier)
- PokéAPI: No limits (but be respectful)
