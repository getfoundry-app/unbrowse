# PokéAPI

All Pokémon data you need.

**Base URL:** `https://pokeapi.co/api/v2`

**Auth:** None required

**Status:** ✅ Verified

## Endpoints

### GET /pokemon
List Pokémon (paginated)

### GET /pokemon/{id}
Get Pokémon by ID or name

### GET /pokemon/{id}/encounters
Get Pokémon encounter locations

### GET /type/{id}
Get type info

### GET /ability/{id}
Get ability info
